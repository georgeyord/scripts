# Git Aliases Cheat sheet

|Alias|Action|
|-----|------|
|br|branch|
|co|checkout|
|ct|commit -m|
|lba|commit graph for all branches including stashes|
|lb|commit graph for current branch|
|lc|commits in one shot|
|la|commits with full details|
|ls|commits with date, description and user|
|pr|pull --rebase|
|st|status|
